# Семейный бюджет (лендинг + калькулятор)

Небольшой статический сайт: приветственный лендинг (`index.html`) и калькулятор семейного бюджета (`калькулятор/калькулятор.html`).

## Как открыть

1. Откройте файл `index.html` в браузере.
2. Нажмите кнопку **«Открыть калькулятор»**.

Калькулятор работает полностью в браузере (данные сохраняются в `localStorage`).

## Что умеет калькулятор

- Расчёт общего дохода, расходов и остатка за месяц
- Процент расходов от дохода
- Прогресс по цели накопления и прогноз “за сколько месяцев”
- Добавление пользовательских категорий расходов
- RU/EN переключение и светлая/тёмная тема
- Автосохранение и экспорт сводки в буфер обмена

## Технологии

- Чистый HTML + встроенный CSS + встроенный JavaScript
